# Pocket Arcade

> A lightweight browser arcade with three instant-play games: Conway's Game of Life, Neon Snake, and Memory Match.

Live site:
`https://toby-qian.github.io/pocket-arcade/`

## Included games

- **Game of Life**: Draw cells, load classic presets, and watch patterns evolve.
- **Neon Snake**: Fast arcade play with keyboard controls and mobile-friendly touch buttons.
- **Memory Match**: A simple tap-to-play card-matching game for quick sessions.

## Why this version is different

- Built as a **user-facing mini game site**, not just a single demo page
- Includes a **guide section**, **game switcher**, and **clear in-page controls**
- Works on both **desktop and mobile**
- Pure **HTML + CSS + JavaScript**, no framework, no build step

## Run locally

```bash
python -m http.server 8000
```

Then open:

`http://localhost:8000`

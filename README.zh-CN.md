# 康威生命游戏

> 浏览器里的零玩家元胞自动机。画几个细胞，点开始，看它们按三条简单规则自顾自地繁殖与消亡。内置高斯帕滑翔机枪、脉冲星等经典图案。

## 功能

- **深色主题**，带网格线的画布，细胞按位置渐变着色
- **环形边界** —— 细胞从一侧滑出会从对侧滑入
- **控制按钮**：开始/暂停、单步、清空、随机填充
- **速度调节**（1–60 fps）与**格子大小调节**（4–30 px）
- **预设图案**：滑翔机 · 高斯帕滑翔机枪 · 脉冲星 · 轻型飞船 LWSS · R-pentomino
- **点击拖动**绘制细胞（首次点击决定画/擦）
- **键盘快捷键**：`空格` 启停 · `S` 单步 · `C` 清空 · `R` 随机
- **实时统计** —— 代数与存活数量
- 纯 Canvas 2D + 原生 JavaScript，无依赖，无构建。

## 规则（B3/S23）

每一代，每个细胞根据它周围 8 个邻居的状态决定命运：

- **存活**细胞周围有 2 或 3 个活邻居 → 继续存活
- **死亡**细胞周围恰好有 3 个活邻居 → 出生
- 其他情况 → 死亡

## 本地运行

直接浏览器打开 `index.html`，或：

```bash
python -m http.server 8000
# 访问 http://localhost:8000
```

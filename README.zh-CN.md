# Pocket Arcade · 口袋街机

> 一个免安装、免登录的浏览器小游戏站，收录五个即开即玩的小游戏：康威生命游戏、霓虹贪吃蛇、记忆翻牌、2048 和扫雷。纯 HTML + CSS + JavaScript，没有构建步骤，没有框架。

**在线体验 →** https://toby-qian.github.io/pocket-arcade/

**语言 / Language:** [English](README.md) · **简体中文**

| 2 | [霓虹贪吃蛇](#2-霓虹贪吃蛇) | 快节奏 · 街机 | 方向键 / WASD · 画布滑动 · 虚拟方向键 |

### 1. 康威生命游戏
在深色环形网格上画细胞，点开始，让三条简单规则自己演化出意想不到的图案。内置**滑翔机**、**高斯帕滑翔机枪**、**脉冲星**、**LWSS**、**R-pentomino** 五个经典预设，可调节速度（1–60 fps）和格子大小（4–30 px）。

### 2. 霓虹贪吃蛇
经典玩法，对手机端做了适配 —— 画布可直接滑动，有虚拟方向键，电脑端支持方向键和 WASD。最高分保存在本地。

### 3. 记忆翻牌
4×4 棋盘上 8 对 emoji 打乱分布，点击翻牌找到所有对子。HUD 实时显示步数、配对数，通关时显示用时。

### 4. 2048
相同数字相撞合并，冲击 2048 或更高。支持**一步撤销**（按钮或 `Ctrl+Z`），本地保存最高分。

### 5. 扫雷
三档难度（9×9 / 12×12 / 16×16）。首次点击保证安全并自动展开。手机端长按标记旗子，会带轻微震动反馈。

## 这个版本的特点

- 当作**对外小游戏站**设计，而不是单个 demo 页
- **共享控制层** —— 键盘事件按当前激活的 tab 分派，避免贪吃蛇页面按空格误触生命游戏
- **移动端优先**：每个交互面按需设置 `touch-action`，顶部/底部考虑了 iPhone 刘海（`env(safe-area-inset-*)`），虚拟方向键去掉了 300ms 延迟
- **画布清晰**：所有 canvas 按 `devicePixelRatio` 缩放，Retina 和手机屏不糊
- **自动暂停**：切换 tab 或浏览器页面隐藏时，贪吃蛇和生命游戏会暂停循环，不在后台烧 CPU
- 无依赖，总共约 20 KB，文件夹复制到哪里都能双击 `index.html` 直接玩

## 本地运行

```bash
# 任意静态服务器都可
python -m http.server 8000
# 访问 http://localhost:8000
```

也可以直接双击 `index.html`，不需要服务器。
